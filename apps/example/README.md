# Example

This is an example [Next.js](https://nextjs.org/) project which uses both `@coinbase/cookie-banner` and `@coinbase/cookie-manager`

## Getting Started

Install dependencies

```bash
npm install
# or
yarn
# or
pnpm install
```

Then run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
